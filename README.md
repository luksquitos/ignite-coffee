## Executando

1. Clonando o repositório


```bash
git clone https://github.com/luksquitos/ignite-coffee.git
```

2. Instalando dependências 

```bash
cd ignite-coffee
npm i
```

3. Iniciando aplicação

```bash
npm start dev
```

## Acessando a aplicação
A aplicação pode ser acessada em `http://localhost:5173`.
